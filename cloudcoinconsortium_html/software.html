Software

Anyone can include CloudCoins into their software

The CloudCoin Consortium was setup to enable the world to use CloudCoins free of charge. We have many developers who share their work with us.

You are free to use this software and modify it so long as you are not creating malware.

If you have any questions or concerns, please email CloudCoin@Protonmail.com

We are working frantically to create great client-side software.
